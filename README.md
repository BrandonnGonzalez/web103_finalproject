## About

### Description and Purpose

UpVendor is a full-stack web application that connects users with unique spaces, services, or experiences offered by independent vendors, similar in concept to Airbnb. Built with React on the frontend and Express.js on the backend, the platform provides a modern, responsive interface styled with Tailwind CSS and enhanced with Mantine UI components and Tabler icons. Hosted on AWS Amplify, UpVendor focuses on simplicity, performance, and scalability while allowing vendors to showcase listings and users to discover, book, and interact with offerings seamlessly.

The purpose of UpVendor is to empower small businesses and independent vendors by giving them a platform to reach a wider audience while providing users with a streamlined way to discover and book unique offerings. It aims to reduce friction in the marketplace by offering intuitive search, transparent pricing, and easy booking workflows, ultimately creating a mutually beneficial ecosystem for both vendors and customers.

### Inspiration

UpVendor is inspired by platforms like Airbnb and modern marketplace applications that prioritize user experience, clean design, and accessibility. The goal is to replicate the simplicity and effectiveness of these platforms while tailoring the experience toward smaller-scale vendors, allowing for more personalized and diverse offerings beyond traditional lodging.

## Tech Stack

Frontend: React (Javascript), Mantine UI, Tailwind CSS

Backend: Express.js
Deployment / Hosting: AWS Amplify, AWS Beanstalk

## Features

### 1. User Authentication

Secure sign-up and login functionality for both customers and vendors, including session management and protected routes.

### 2. Vendor Listings Management

Vendors can create, edit, and delete listings with details such as title, description, pricing, images, and availability.

### 3. Search & Filtering

Users can search for listings and apply filters such as price range, category, or availability to quickly find relevant results.

### 4. Booking System

Users can reserve listings by selecting dates or time slots, with basic validation and confirmation flow.

### 5. User Profiles

Each user has a profile page displaying their personal info, bookings, and (for vendors) their active listings.

### 6. Reviews & Ratings

Users can leave ratings and reviews on listings they’ve booked, helping build trust and transparency.
