# User Stories

Reference the Writing User Stories final project guide in the course portal for more information about how to complete each of the sections below.

## Outline User Roles

- Guest (unauthenticated user)
- Customer (authenticated user)
- Vendor (authenticated seller)

## Draft User Stories

[👉🏾👉🏾👉🏾 Include at least at least 10 user stories in this format:]

1. Guest
As a guest, I want to browse listings without creating an account so that I can explore available options before signing up.

2. Guest
As a guest, I want to search for listings by keyword so that I can quickly find relevant offerings.

3. Customer
As a customer, I want to create an account and log in so that I can book listings and manage my activity.

4. Customer
As a customer, I want to book a listing so that I can reserve a service or space.

5. Customer
As a customer, I want to view my booking history so that I can keep track of my reservations.

6. Customer
As a customer, I want to leave reviews on listings so that I can share my experience with others.

7. Vendor
As a vendor, I want to create a listing so that I can offer my service or space to users.

8. Vendor
As a vendor, I want to edit or delete my listings so that I can keep my offerings up to date.

9. Vendor
As a vendor, I want to view bookings made on my listings so that I can manage my schedule.

10. Vendor
As a vendor, I want to receive notifications when someone books my listing so that I can respond or prepare accordingly.
